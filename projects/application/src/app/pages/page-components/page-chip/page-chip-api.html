API reference

Directives

CbChip

Selector: cb-chip

Exported as: CbChip

Name Description
@Input() bg: boolean; Set if bg is true or false.
@Input() icon: string; Set icon to chip.
@Input() size: string; Size of the chip: small, medium (default), large
@Input() color: string; Color of the chip: blue, green, orange, green-light green-lighter

CbChipList

Selector: cb-chip-list

Exported as: CbChipList

Name Description
@Input() direction: string; Direction of list: vertical, horizontal (default)
