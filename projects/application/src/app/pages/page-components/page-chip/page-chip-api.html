CbChip

A cubo chip component

Selector: cb-chip

Exported as: CbChip

Name Description
@Input() image: string; The link for the chip's image.
@Input() name: string; An chip can be created with the name initials instead of an image.
@Input() size: string; Size of the chip: xsmall, small, medium, large, xlarge
