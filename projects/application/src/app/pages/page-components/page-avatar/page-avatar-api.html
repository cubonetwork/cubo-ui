CbAvatar

A cubo avatar component

Selector: cb-avatar

Exported as: CbAvatar

Name Description
@Input() image: string; The link for the avatar's image.
@Input() name: string; An avatar can be created with the name initials instead of an image.
@Input() size: string; Size of the avatar: xsmall, small, medium, large, xlarge
